### Spree
**Default admin**
```
Email: spree@example.com
Password: spree123
```

**Rake to create admin**
```
$ rake spree_auth:admin:create
```

### Scraper
**Install node.js libraries**
```
$ cd scraper
$ npm install
```

**Run the program with output file name**
```
$ node scraper.js > output.csv
```
